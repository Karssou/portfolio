## Why this project?

This project was born out of my passion for the series **Arcane** and my desire to push the boundaries of what I could achieve using only **HTML**, **SCSS**, and **JavaScript**.

At the time, I wanted to design an immersive experience capable of capturing the visual identity of the show through animations, transitions, and a carefully crafted layout.

This project represents one of my first large-scale front-end achievements.

## Goals

The objectives were mainly focused on the user experience:

- Reproduce a visual identity faithful to the Arcane universe.
- Create smooth and immersive navigation.
- Experiment with animations using GSAP.
- Develop a fully responsive website.
- Deepen my knowledge of HTML, SCSS, and JavaScript.

## Architecture

The site was built without any framework, using only:

- HTML for the structure.
- SCSS for styling organization.
- JavaScript for interactions.
- GSAP for animations.

The entire project is organized in a modular way to separate the different interface components and make future updates easier.

## What I learned

Looking back, this project no longer reflects my current development practices.

However, it represents an important milestone in my learning journey. It allowed me to discover the basics of web animation, improve my CSS skills, and understand the importance of user experience in interface design.

I intentionally keep this project in my portfolio, as it stands as a testament to my growth and the progression of my skills over the years.
